General Settings

Unless specified otherwise, 0 = false, 1 = true

  • save_workspace_on_close - save workspace settings on close (if this is set to 0, then you must use the Workspace Save menu to save the current workspace state) (default is 1)
  • widgetstyle - set the style of the editor (Qt style is used)
    • options are: windows, motif, cde, plastique, windowsxp, macintosh
    • default is plastique
    • leave blank to use OS default ("widgetstyle":"")
  • style - path to the Qt stylesheet to use
  • showPath - show the full file path in the top right (default is 1)
  • activeEditors - the list of active editors that are enabled
  • visibleLineTracking - event that tracks what lines are visible (used by the outline plugin) (default is 1)
  • checkFileChanges - the number of seconds to check if a file has changed (set to 0 if you don't want to check for file changes)
  • network - proxy - set the proxy server for any plugins that pull from the web (leave blank for no proxy server)

Window Settings

  • openMode - size the editor opens as
    • 1 = custom - use size and margin attributes
    • 2 = maximize window
  • margin - if openMode = 1, specify the margins for the editor
  • size - size of the editor in pixels or percent (add % at end).  The size will subtract off the margins
  • Window Switcher
    • view_folder - view parent folder name in tab along with filename (default is 0)
    • height - height of the window switcher (default is about 3 rows of tabs)
  • pluginLeft, pluginBottom, pluginRight
    • visible - plugin is visible by default
      • 1 = visible
      • 0 = hidden by default (can be made visible through the Main Menu > Window menu
    • width/height - specify default width or height the plugin window is opened to
    • showTabText - show the title of the tabs for the plugins (set to 0 to only show icons)
