Plugins

  • Scope comes installed with some plugins by default

Enabling/Disabling Plugins

  • Enable/Disable plugins by double clicking in the Enabled column
    • Y = enabled
    • blank = disabled
  • Plugins will attempt to load when toggled to enabled, but a restart may be needed to fully load the plugin
  • Disabling a plugin will not take affect until restarting Scope

Plugin Settings

  • Click on the plugin name to view any plugin specific settings below the plugin list
  • The plugin settings are in a json format
  • Be sure to click Save after modifying the plugins

Adding Plugins

  • Plugins can installed by using the Install buttons:
    • File - a zip file on your computer
    • URL - a url to a zip file
    • Manual - copy the plugin folder to <scope>/plugins then add with this button
  • For the File and URL methods, the plugins must be zipped
  • Plugins must be in a particular format to actually get loaded
    • See <scope>/plugins/_template for an example
    • Delete the scope_editor.py file if creating a plugin and not another editor
