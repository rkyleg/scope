The format for these settings are json
 

Standard keys are:

  • editor - the editor to use for the language
  • run - the command to execute for this language (filepath is appended to the end of the command)
    • use the prefix "preview " followed by a command to generate html to generate html and show it in the preview pane
  • fave - set to 0, if you don't want this language to show up in the favorites lists
  • theme - for ace editor, this will change the theme
  • wordwrap - 0 for no wordwrap, 1 for wordwrap
