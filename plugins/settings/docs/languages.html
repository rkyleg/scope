The format for these settings are json
 

Standard keys are:

  • editor - the editor to use for the language
  • run - the command to execute for this language
    • filepath is appended to the end of the command
    •   or use {{filename}} to specify where to insert the filename in the command
          Example: python -u {{filename}}
    • use the prefix "preview " followed by a command to generate html to generate html and show it in the preview pane
          Example: preview /home/home/node/markdown.sh
  • compile
    • new_ext - the extension to use for the new file created by the command (optional)
    • cmd - the command to run
      • {{filename}} - use within the command to insert the filename
      • {{new_file}} - use within the command to insert the new filename with the extension
    Example: "compile":{"new_ext":"min.js","cmd":"uglifyjs {{filename}} -o {{new_file}}"}
  • fave - set to 0, if you don't want this language to show up in the favorites lists
  • theme - for ace editor, this will change the theme
  • wordwrap - 0 for no wordwrap, 1 for wordwrap
