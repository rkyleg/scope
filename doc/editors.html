Home | Editors

Armadillo comes with multiple code editors, so you can choose the one you want for each language.

  • Ace - Ace is an embeddable code editor written in JavaScript. It matches the features and performance of native editors such as Sublime, Vim and TextMate.
  • CKEditor - CKEditor is a ready-for-use HTML text editor designed to simplify web content creation. It's a WYSIWYG editor that has common word processor features to generate html.
  • QScintilla - QScintilla is a port to Qt of Neil Hodgson's Scintilla C++ editor control. As well as features found in standard text editing components, QScintilla includes features especially useful when editing and debugging source code.
