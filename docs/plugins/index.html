Plugins

Scope comes with standard plugins to the left and bottom of the editor window. Custom plugins can be created with Python and PyQt

Standard Plugins

  • File Browser - Browse files

  • Outline - Outline of code structure

  • Find/Replace - Common find/replace functions

  • Output - Runs code in separate process and displays output.

  • Python Console - An interactive Python shell

  • Snippets - view/store snippets of code

    • You can set an existing directory of snippets in the Settings - under Plugins > Snippets

      "path":"/home/username/scripts"

  • Preview - a built in WebKit webbrowser for previewing html and markdown. A file does not need to be saved to use the preview.

  • PyQt Converter - Convert Qt Designer ui files to python. This plugin is not active by default

  • Scratchpad - A simple textbox for temporarily storing text/code. currently this is not saved when Scope closes

Select Active Plugins

To specify which plugins are active, toggle them on and off in the Plugins section of the Settings

Custom Plugins

You can create your own plugins with Python and PyQt.

  1. Create a new directory for your plugin in /scope/plugins
  2. Look at the plugins/_template for a basic example
    • For a plugin you need the plugin.json and scope_plugin.py files (can remove the scope_editor.py file)
    • To add an editor you need the scope_editor.py file (remove the plugin.json and scope_plugin.py files)
