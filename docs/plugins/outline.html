Outline Plugin

  • The Outline plugin outlines the code structure to help with organization
  • Double click on outline text to goto that section of code
  • Special comment formatting will add that comment into the outline as a header
  • Utilizes existing spacing in code to format outline
  • The Outline updates by one of the following methods
    • Saving the file
    • Ctrl+O
    • right click and select update
    • (there is a setting in the plugins section to update in realtime: alwaysUpdate=1 )
  • The current visible code in the editor is highlighted

Current Supported languages

  • Python
  • JavaScript
  • CSS
  • HTML
  • Markdown
  • ini/config
  • handlebars/mustache
  • SQL

Comment Protocol

To add a comment to the outline, you typically add 3 dashes (---) after the comment character.
  • Python: #---Comment
  • JavaScript: //---Comment
  • CSS: /*---Comment*/
  • HTML: <!---Comment-->
  • Markdown: (not available)
  • ini/configobj #---Comment
  • Handlebars/Mustache: same comments as HTML
  • SQL: ----
