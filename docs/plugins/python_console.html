Python Console

The Python Console is a simple interactive Python interpreter with access to the Scope environment.

Notes

  • Unless you are running from Python source, the shell uses Python 2.7

  • If you don't have Python 2.7 installed and are using the Windows binary, then you will have limited Python functions - only what is bundled with Scope.

  • Ctrl +L - will launch the shell outside of Scope with your default Python install (Python and PyQt must be installed).

  • You can access Scope from the shell with the scope object. Be careful as you can lose work.

      scope.currentWidget() # Gets the current editor widget
  scope.currentWidget().getText() # Gets the text from the current editor widget
